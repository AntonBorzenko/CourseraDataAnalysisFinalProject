## How to launch
1. Clone repository

```bash
git clone https://github.com/AntonBorzenko/DataAnalysisFinalProject
```

2. Run server.py
```bash
cd DataAnalysisFinalProject/src/
python3 server.py
```
3. Open [http://localhost:5000](http://localhost:5000)
